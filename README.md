# 타일의 정의된다

각 타일은 위, 오른쪽, 아래, 왼쪽 순으로 연결(1)되거나 단절(0)되는 것이 정의된다.

해당 정의내역은 메인 함수에 `Tiles` 로 정의된다. 해당 갯수에 맞추어 WFC를 이용하여 화면에 도식을 그린다.
